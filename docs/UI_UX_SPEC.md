# Especificação de UI/UX

## Princípios de Design
- **Identidade Visual:** Seguir estritamente o arquivo `IV_DESIGN.md`. (Aguardando definição).
- **Mobile First:** Interface otimizada para navegadores de smartphone.

## Componentes da Interface
1. **Grid de Produtos:** Molduras que representam a vitrine real, exibindo:
   - Nome do Artesanato.
   - Localização (Ex: "Mola A1").
   - Preço Unitário.
   - Seletor de quantidade (limitado ao estoque atual).
2. **Sacola Flutuante:** Resumo do valor total visível o tempo todo.
3. **Modal de Pagamento:**
   - Exibição clara do Pix Copia e Cola.
   - Botão "Copiar Código".
   - Feedback visual de "Aguardando pagamento...".
4. **Animação de Sucesso:** Gatilho visual imediato pós-confirmação, seguido de instrução: "Retire seus produtos abaixo".
